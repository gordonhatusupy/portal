# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-01

- Initial public release
- Native macOS app built with SwiftUI + AppKit
- Menu bar status item with floating server panel
- Detects running local web servers and infers project names
- Shows Git branch, full localhost URL, and runtime
- Supports opening a server in the browser
- Supports stopping a server from the panel
- Includes an empty state and polished native macOS UI
- Dock-visible while running with standard macOS active-state behavior
